# TechBuild Group

A minimal React + Vite starter for the "TechBuild Group" website.

Getting started (PowerShell / Windows):

```powershell
cd "c:\Users\simbizi\Downloads\document of jesai\projects\Techbuild"
npm install
npm run dev
```

Open the shown local URL (usually http://localhost:5173) in your browser.
